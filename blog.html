Bootstrap Library for Mobile Development
Feb. 28 2021

Let's Go!

Introduction

Everyone wants to have a good-looking website to represent them on the internet, right? In this modern day and age, first impressions are everything. That’s why having a presentable and user-friendly website is very important to retain user attention as well as make things as convenient as possible. On top of that, the world is shifting towards a more mobile-diverse tech ecosystem with more and more products being released into the smartphone market. That being said, the rise of the mobile market calls for more websites to be designed to look and feel good.

The topic I wish to do my research on is the mobile development aspects of the Bootstrap framework which focuses on developing mobile-first web applications. The term mobile-first is defined as websites that can function and look good on a desktop-based environment, but also function and look good on a mobile-based environment. Bootstrap is a free and open-source front-end framework that contains CSS and JavaScript-based design templates for interface components that assist in making websites look great and work on both desktop and mobile environments.

Bootstrap was developed by Mark Otto and Jacob Thornton. The duo were employees at Twitter and were frustrated at the inconsistencies and high maintenance burdens that the interface development team had to deal with when it came to interface development, so they developed a framework called Twitter Blueprint to develop a front-end framework that unifies and brings consistency to the team.

Why Bootstrap?

Of all the frameworks out there, why Bootstrap? There are so many different libraries to choose from but why specifically Bootstrap? My reasoning is because Bootstrap is THE most popular CSS framework out there with over ~~% of GitHub repositories utilizing Bootstrap, that's a lot! I chose Bootstrap because I do have some prior knowledge on it although very basic and I want to expand it and I saw that this Emerging Trends assignment would be a good opportunity to do so! I found that Bootstraps integration process, while very similar to other frameworks, is crazy easy and is as simple as a simple download and import. Then utilizing Bootstrap to its fullest potential is even easier. I think what makes Bootstrap so popular is because of how beautiful its components are as well as how simple and easy it is to implement into your HTML or CSS code.

Learning Process

Learning how to use Bootstrap was actually quite simple. A few YouTube videos and a quick glance at the documentation was all I needed to get started with Bootstrap! I used sites such as this and this as documentation guides to help me understand what Bootstrap has to offer, then I watched videos such as this to help put it all together. I created a new GitHub repository and then converted it into something called a GitHub Page, which is essentially GitHub's free service of allowing users to host their own website using the HTML code stored in the repository. From there, I ran the usual downloads and imports to get Bootstrap installed into my project and away I went! Implementing one thing after another, testing things out as I went, it was a long and fun learning process. The fun part of learning Bootstrap was reading the official documentation and trying out the different components that Bootstrap has to offer. Seeing everything show up as save and open the HTML file has a certain rewarding feeling to it, as if you're seeing the fruits of your labour.

Once I got comfortable with all the different components and design templates, I began to start adding a bit of cohesion to my project, I wanted to formulate an actual website and make it look nice. The nice thing about Bootstrap is that every single component in its library is optimized for mobile platforms, so while my project will look nice on desktop, it will also function and look perfectly fine on mobile as well! (Try opening this website on your phone!)

Implementing everything was so easy because of Bootstraps column and grid system, which is Bootstraps way of breaking down a webpage into smaller squares that can be filled in with components or text, which is utilized to make the desktop webpage look beautiful on a mobile platform too. The grid system organizes components in such a way that it utilizes as much space as the website gives as possible to maximize screen space, and when the screen space gets smaller for whatever reason such as scaling the webpage down or viewing it on a mobile phone, the grid system reorganizes itself to utilize the limited space as efficiently as possible, which makes it possible for the desktop webpage to look so good on a smaller screen.

With that knowledge in mind, I had no doubt that whatever I was doing to the desktop webpage would look absolutely amazing on a mobile platform.

Results

The results of what I learned can be seen with this website you are reading this very blog post on! Everything on this website was created using Bootstrap and every component is a variation of a component given to me from the official documentation. Before Emerging Trends, I had this website created before as a personal project but it didn't have much on it. After being given the opportunity to go back and update this project, I was able to take this opportunity and add my blog posts to this site as a way to present my findings. I am quite proud of what I have accomplished so far and hope to further my research so that I can carry it over into the real world.
